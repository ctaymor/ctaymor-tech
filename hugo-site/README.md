Dependencies:
Runtime:
* `minimal` hugo theme

Dev:
* `sass`
* `gcloud sdk`
* `imagemagick`
* `hugo`

To Build:

To build hugo sass files: `make sass`

To Serve locally:
`hugo serve`

Add a -D flag to include draft posts.

To Deploy (or just git push and travis will handle it):
```bash
cd ctaymor-home
gcloud app deploy --project ctaymor-home
```
