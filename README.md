Dependencies:
Runtime:
* `minimal` hugo theme

Dev:
* `sass`
* `gcloud sdk`
* `imagemagick`
* `hugo`

To Build:

To build sass files, `sass www/css/index.scss www/css/index.css`

To Deploy:
```bash
cd ctaymor-home
gcloud app deploy --project ctaymor-home
```
